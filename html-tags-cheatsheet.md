# HTML Tags Cheatsheet (HTML Living Standard)

Notes:
- Most elements are **not** self-closing. The **void elements** listed below have no end tag.
- Many elements are **obsolete** (deprecated) and should not be used in new pages.

- `<!doctype html>`
- `<html>`
- `<head>`
- `<title>`
- `<base>`
- `<link>`
- `<meta>`
- `<style>`
- `<body>`
